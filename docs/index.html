Reporting a bug

Please include:

  • Your iPhone model and iOS version
  • What you were doing (importing a game, playing, running the Windows 98 install wizard, …)
  • What you expected, and what happened instead
  • For game problems: the game's format (.jsdos or .zip) and, if you know it, which DOS setup it needs (sound card, memory)

Windows 98 install wizard (Experimental)

A full install runs unattended for roughly 30–60 minutes, and the feature is young — that's why it's labeled Experimental. If an install fails, it fails cleanly: nothing is lost and you can simply try again. When reporting a wizard problem, mention roughly how far it got (the stage text on the progress screen).

PocketDOS ships no Microsoft software. The wizard requires your own Windows 98 SE CD image and your own product key; both are processed entirely on your device and never leave it.

Frequently asked

Where are my games stored?
In the Files app under On My iPhone → PocketDOS → Games. You can back them up, add to them, or remove them from there directly.

Does PocketDOS collect any data?
No. There's no telemetry, no analytics, no account, and the emulator can't reach the network. See the privacy policy.

Can I get my saves onto a new phone?
Small save files sync automatically through your own iCloud Drive if iCloud is enabled. Game files themselves can be copied via the Files app.

Purchase or App Store problems?
Use Apple's Report a Problem.
